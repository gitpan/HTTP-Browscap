Description of Contribution:
                A module (suggested name HTTP::Browscap) that provides information
                on a web browsers' capabilities (eg - table support, frame support),
                the browser being identified by its "User Agent" string. This will 
                use the existing Microsoft Browscap.ini database, a version of 
                which is freely downloadable and actively maintained at 
                http://www.browserhawk.com/browscap/
